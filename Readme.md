# 🛍 Proyecto Tienda de Ropa - MongoDB

## 📌 Descripción del Proyecto

Este proyecto consiste en el desarrollo de una base de datos no relacional utilizando MongoDB para la gestión de una tienda de ropa.

La base de datos permite administrar:

- Usuarios
- Marcas
- Prendas
- Ventas

Se implementan operaciones CRUD (Create, Read, Update, Delete) para cada colección, así como consultas avanzadas utilizando el framework de agregación (aggregate).
